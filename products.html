ECOSYSTEM PRODUCT

StackPilot

Multi-Account AWS Infrastructure Console

StackPilot provides platform engineering and developer teams with agentless, read-only visibility into multi-account cloud topologies, active resource inventories, and security compliance.

View on GitHub Documentation

Why It Exists

Modern cloud infrastructure grows complex in seconds. As teams spin up multiple accounts and Kubernetes clusters, they lose track of what resources actually exist. Traditional observability solutions require deploying heavy agents, exposing security credentials, and paying high SaaS fees. StackPilot solves this by offering a zero-install, 100% read-only interface that scans and maps entire topologies in real time.

Core Features

🔍

Resource Discovery

Agentless scanning and inventory of AWS resources across all active regions and linked accounts.

🧠

Cloud Intelligence

Maps real-time topology, dependency relationships, and provides Google-like universal search.

🛡️

Governance & Security

Tracks compliance scoring, drift, and maps vulnerabilities safely using read-only access.

📡

Operational Visibility

Consolidates EKS nodes, CPU allocations, and active container runtimes in a single pane.

Architecture

Agentless Metadata Scanner

Leverages native cloud and Kubernetes APIs directly without deploying agents or sidecars in your production environments.

Read-Only Cache Layer

Maintains a local Redis-backed cache to speed up universal search queries and minimize API rate-limiting penalties.

Interactive React Console

A modern client-side React UI dashboard designed for rapid navigation, VPC topology rendering, and resource filtering.

Specifications, How it Works & FAQ

Execution Stack (Technology) React / Java / Go / Redis / AWS SDK
How It Works (Step 1) Delegate a read-only AWS IAM role or connect local Kubernetes configs.
How It Works (Step 2) StackPilot metadata scanner maps linked accounts and queries resources.
How It Works (Step 3) Instantly browse resource inventory, topology graphs, and cost allocations.
Licensing Model Apache-2.0 License (Fully open-source repository and configuration Helm charts)
Product Roadmap Multi-cloud support (GCP & Azure), automated config drift notifications, OpenTelemetry exporter integration.
FAQ: Is it safe for production? Yes. StackPilot operates with read-only IAM access policies. It never mutates or writes state.
FAQ: Does it require agents? No. It relies entirely on API-level scanning and queries. No daemons are installed on your hosts.
