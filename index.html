ABOUT SILICON ROOTS

An engineering organization, not a single product.

Silicon Roots is a small engineering organization building open infrastructure software for the teams who operate modern cloud systems. We build because we have lived the problem: infrastructure grows faster than the tools used to understand it.

Why we build software

Cloud platforms have become distributed systems of their own—spread across accounts, regions, clusters and teams. We build software that turns that sprawl into something engineers can actually reason about, govern and operate with confidence.

EVALUATE STACKPILOT

Get Started in 60 Seconds

StackPilot is open source and self-hostable. Run it locally or deploy it to your Kubernetes cluster.

Option 1: Docker Compose

Perfect for local evaluation and testing discovery flows.

bash 
# Clone the repository
git clone https://github.com/siliconroots/stackpilot.git
cd stackpilot

# Boot up the local database and platform services
docker-compose up -d

# Open StackPilot Console in your browser
echo "http://localhost:3000"

Option 2: Kubernetes (Helm)

Deploy a scalable instance inside your EKS/K8s infrastructure.

bash 
# Add the Helm repository
helm repo add siliconroots https://charts.siliconroots.io
helm repo update

# Install StackPilot under the platform namespace
helm install stackpilot siliconroots/stackpilot \
  --namespace platform \
  --create-namespace \
  --values values.yaml
